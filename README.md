# vue-input-keyboard
A keyboard of input with validator, for: numbers, telephone, IDs...  以数字为主的键盘模拟组件，且内置有效性验证，优化输入互动逻辑，提高用户的输入效率。


## DEVELOP

* init
```
    npm install
```
* code & build
```
    npm run build
```
* publish the code to npm if you want to do it.
```
    npm config set registry=http://registry.npmjs.org

    npm publish
```
